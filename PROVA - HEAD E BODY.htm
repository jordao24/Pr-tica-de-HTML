PROVA 1 - O que é HTML, HEAD e BODY?

O que é o HTML

O HTML é uma linguagem de marcação, isso é, uma linguagem estrutural, que dará a estrutura inicial para um site na internet. É através dele que se estabelece as bases que outras linguagens de programação farão funcionar, sendo o CSS a parte estética e visual, e o JavaScript como a parte funcional.

O HTML portanto tem a função principal de definir a estrutura de um site e dar as informações necessárias para que este possa ser incluso na web geral.

O que é o HEAD?

O head de um documento HTML será responsável por definir a base do documento extratextualmente. Isso é, dentro da head se encontrará funções meta, title, que serão responsáveis por definir o tipo de caractere, a descrição, links para scripts de CSS e JavaScript e também o titúlo do texto.

O que é o BODY?

O body é a parte visível do texto, sendo a que estou lendo agora. Nessa será incluso os elementos de construção textual, como a heading (de 1 a 5) e os parágrafos, com elementos como o itálico e o negrito.
